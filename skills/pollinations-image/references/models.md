# Pollinations Image Models

## Available Image Models

| Model | Description | Best For |
|-------|-------------|----------|
| `zimage` | Default model, good balance of quality and speed | General purpose |
| `flux` | High-quality flux model | High quality images |
| `turbo` | Fast generation | Quick previews |
| `gptimage` | GPT-based image generation | Complex compositions |
| `seedream` | Seedream model | Artistic styles |
| `nanobanana` | Creative model | Experimental |
| `nanobanana-pro` | Pro version of nanobanana | Professional |

## Size Recommendations

| Use Case | Width | Height |
|----------|-------|--------|
| Default | 1024 | 768 |
| HD/Large | 1600 | 1200 |
| Square | 1024 | 1024 |
| Portrait | 768 | 1024 |
| Wide | 1600 | 900 |

## Model Selection Tips

- **Quick test**: Use `turbo` for fast results
- **General use**: `zimage` is a good default
- **High quality**: `flux` or `gptimage`
- **Artistic**: `nanobanana` or `seedream`
