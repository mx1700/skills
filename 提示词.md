## 文生图

创建一个 agent skill
功能是 使用 https://enter.pollinations.ai/api/docs#tag/%EF%B8%8F-image-generation/GET/image/{prompt} 的文生图接口, 生成图片
可接受参数:

- prompt 提示词
- model 模型 默认 zimage
- width 宽度 1024
- height 高度 768

当用户要求大尺寸/高清图片时, 生成 1600×1200 尺寸图片
创建 .env 文件用以配置 API_KEY
脚本代码里检查 .env 是否配置正确, 如果配置不正确给出提示
AI收到提示后,要求用户提供 API_KEY, 然后由AI生成/更新 .env 文件
调用接口生成图片后,保存到本地,默认保存到临时文件夹
